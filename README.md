# JiraPythonTool

A small Python CLI to interact with Jira Cloud (starter scaffold).

Basic features:
- CLI to fetch an issue summary
- Lightweight `JiraClient` using `httpx`

Quick start:
- Set environment variables `JIRA_BASE_URL`, `JIRA_EMAIL`, `JIRA_API_TOKEN` or pass options to the CLI
- Install: `pip install -e .`
